本工具支持Pal的DOS版本与Windows95版，请诸君放心食用~
请直接将工具“SDLPalStudio_x32_Debug.exe”放入游戏目录中运行即可~

1、DOS版文本暂未从 Big5 编码码转换为 GB2312 编码，故解档后文本乱码属正常现象~
2、Object解析暂时只支持原版，改版未支持，后续将对其做相应支持的Cfg翻译表文件~
3、若运行时报 Not Found xxx.DLL 请尝试直接将 FIXME 目录下对应系统位数的补丁与
     工具一同放入游戏目录中，如果还是不行，请以 管理员身份运行 对应位数的
     “xx位系统安装.bat”安装该补丁以解决此问题......
